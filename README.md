## 📝 使用指南

### 记录原则
- ✅ 及时记录，不要拖延
- ✅ 结构清晰，方便检索
- ✅ 包含实例，理论结合实践
- ✅ 定期回顾，持续更新

### 命名规范
- 文件命名：`主题-日期.md` 或 `主题-说明.md`
- 日期格式：`YYYY-MM-DD`
- 使用有意义的描述性名称

### 标签系统
在笔记中使用标签便于分类和搜索：
- `#编程语言` `#框架` `#工具`
- `#问题解决` `#最佳实践`
- `#进行中` `#已完成` `#待深入`

## 🚀 快速开始

1. **创建新笔记**：在相应分类目录下创建 `.md` 文件
2. **使用模板**：参考 `templates.md` 中的模板格式
3. **定期整理**：每周回顾并整理笔记，保持仓库整洁
