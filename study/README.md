# Study

This folder contains study notes, learning resources, and educational materials across various subjects.

## Example Topics

- Course notes
- Book summaries
- Learning resources
- Study techniques
- Programming concepts
- Language learning
- Online course notes

## How to Use

Organize your study materials by subject or course. For example:
- `programming/` - Programming language notes
- `mathematics/` - Math concepts and formulas
- `languages/` - Language learning notes
- `book-notes/` - Summaries of books you've read

## Template

```markdown
# [Subject/Topic]

**Date**: YYYY-MM-DD
**Source**: [Book/Course/Website]

## Overview

[Brief description]

## Key Concepts

### Concept 1
[Explanation]

### Concept 2
[Explanation]

## Examples

[Code snippets or examples]

## Practice/Exercises

- Exercise 1
- Exercise 2

## Resources

- [Resource 1](url)
```
