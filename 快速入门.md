# 📌 快速入门指南

欢迎使用你的知识记忆库！这里是一些使用建议，帮助你快速上手。

## 🎯 第一步：选择合适的分类

根据要记录的内容类型，选择对应的目录：

| 内容类型 | 存放位置 | 示例 |
|---------|---------|-----|
| 学习某个编程语言 | `01-编程技术/` | Python装饰器用法 |
| 看书或课程笔记 | `02-学习笔记/` | 《代码整洁之道》读书笔记 |
| 项目开发问题 | `03-项目经验/问题解决/` | Django ORM N+1查询问题 |
| 工具配置 | `04-工具资源/开发工具/` | VSCode快捷键配置 |
| 突然的想法 | `05-灵感想法/` | 开发一个时间管理工具 |
| 每天的学习 | `06-日常记录/每日总结/` | 2025-11-11学习总结 |

## ✍️ 第二步：创建笔记文件

### 方式一：使用模板
1. 打开 `templates.md` 查看模板
2. 复制对应的模板内容
3. 在目标目录创建新的 `.md` 文件
4. 粘贴模板并填写内容

### 方式二：自由格式
直接在目标目录创建 `.md` 文件，自由书写。

## 💡 第三步：好的笔记习惯

### 每日记录（推荐）
在 `06-日常记录/每日总结/` 下创建文件：
```
2025-11-11.md
2025-11-12.md
```

简单记录：
- 今天学了什么
- 有什么收获
- 遇到什么问题
- 明天的计划

### 技术笔记
学习新技术时，在对应目录记录：
- 核心概念
- 代码示例
- 实践经验
- 参考资源

### 问题记录
遇到Bug时立即记录：
- 问题现象
- 解决过程
- 最终方案
- 经验总结

## 🔍 第四步：检索和回顾

### 使用 Git 搜索
```powershell
# 搜索关键词
git grep "关键词"

# 查看历史记录
git log --oneline
```

### 定期回顾
- 每周回顾：整理本周笔记，归类和补充
- 每月回顾：总结学习进展，调整计划
- 每季度回顾：复习重要知识点，查缺补漏

## 📦 示例：创建第一条笔记

让我们创建一个示例笔记：

**文件位置**: `06-日常记录/每日总结/2025-11-11.md`

**内容示例**:
```markdown
# 2025-11-11 学习总结

## 今日学习
- [x] 创建了个人知识记忆库
- [x] 学习了 Git 基础命令
- [ ] 开始学习 Python 基础

## 今日收获
1. 建立了系统的知识管理体系
2. 了解了如何使用 Git 进行版本控制
3. 明确了接下来的学习方向

## 遇到的问题
Q: Git 推送到 GitHub 时需要身份验证
A: 使用 Git Credential Manager 进行身份验证

- 完成 Python 基础语法学习
- 记录 Python 学习笔记到仓库
- 练习 Git 基本操作
```

## 🎨 进阶技巧

### 使用图片
将图片放在笔记同级目录的 `images/` 文件夹中：
```markdown
![说明文字](./images/example.png)
```

### 使用代码块
\`\`\`python
def hello():
    print("Hello, Memory!")
\`\`\`

### 使用清单
- [ ] 待完成任务
- [x] 已完成任务

### 添加链接
- 内部链接：`[Python笔记](../01-编程技术/Python/基础语法.md)`
- 外部链接：`[GitHub](https://github.com)`

## ⚡ 同步到 GitHub

定期将笔记推送到 GitHub 备份：

```powershell
# 查看修改
git status

# 添加所有更改
git add .

# 提交更改
git commit -m "添加今日学习笔记"

# 推送到远程
git push
```
